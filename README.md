# Conversor de Monedas - Java

Proyecto desarrollado para el Challenge de Oracle Next Education y Alura Latam.

## Descripción

Este proyecto es un conversor de monedas que utiliza una API de tasas de cambio en tiempo real.

El usuario puede elegir diferentes tipos de conversión entre monedas desde la consola.

## Tecnologías utilizadas

- Java
- ExchangeRate API
- Gson
- IntelliJ IDEA

## Funcionalidades

Conversión entre:

USD → ARS  
ARS → USD  
USD → BRL  
BRL → USD  
USD → COP  
COP → USD
